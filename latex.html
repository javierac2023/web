Sintaxis de las fórmulas matemáticas

  1. Para limpiar el área de entrada se puede escribir $li$ el botón $C$
  2. Para la multiplicación de elementos no es necesario agregar $*$, la calculadora lo interpreta automáticamente.
  3. Las funciones elementales que necesitan un argumento requieren abrir y cerrar paréntesis de manera obligatoria.
  4. Para usar raíz cuadrada se escribe $√(25)$
  5. Para escribir $sin(2)$ se usa sin(2), $cosh(1)$ se escribe cosh(1), etc.
  6. La función logaritmo natural $Ln(14)$ se escribe Ln(14)
  7. El logaritmo de $a$ en base $n$ , $log_n(a)$ se escribe Logn(a)
  8. Las constantes especiales $‘‘\pi "$, $‘‘e"$ se escriben π y e, aunque también es válido colocar $‘‘pi"$.
  9. La función $Atan2(x,y)$ devuelve el arco tangente de las coordenadas $‘‘x"$ e $‘‘y"$ especificadas. El arco tangente es el ángulo medido desde el eje x hasta la línea que contiene el origen $(0 ; 0)$ y el punto con las coordenadas $(x;y)$. El valor del ángulo se expresa en radianes entre $-\pi$ y $\pi$, excluyendo $\pi$ y $-\pi$.
  10. El último valor calculado luego de presionar "=" o enter se almacena en ANS, para almacenar un valor en memoria se usa MS
  11. Para usar potencias como $5^2$ , se usa $5$^$2$ o también $5**2$
  12. El comando convert permite cambiar de base, por ejemplo cambiar 140 en base 10 a base 5, $convert(140,5)$
  13. Para usar combinatoria, por ejemplo $\binom{n}{r}$ se escribe $comb(n,r)$
    14. Funciones Trigonométricas: Cotangente: cot(x) Secante: sec(x) Cosecante: csc(x) Funciones Hiperbólicas: Seno hiperbólico: sinh(x) Coseno hiperbólico: cosh(x) Tangente hiperbólica: tanh(x) Cotangente hiperbólica: coth(x) Secante hiperbólica: sech(x) Cosecante hiperbólica: csch(x)
